1. Welcome

Notion Vault CLI is a simple program that remembers all your passwords for you. You talk to it like a person — just type save my Instagram password — and it stores everything encrypted on your computer only.

No cloud. No accounts. No monthly fee. It works completely offline, even without internet.

Built for normal people. You don't need to be technical. If you can type a sentence, you can use it.

2. Why You Need It

  • Stop reusing passwords. Using one password everywhere is like having one key for your house, car, and office.
  • Stop writing passwords on paper. Paper gets lost. Paper gets seen.
  • Find any password in 2 seconds. Just ask: what's my wifi password
  • Keep work, school, and personal separate. No more mixing your university LMS with your bank login.

3. How It Works (Simple)

Think of it like a locked filing cabinet with three drawers:

🗄️ Personal

Instagram, Gmail, Netflix

🎓 Education

LMS, university portal, student email

💼 Business

Work email, AWS, Slack

You have one master key (your master password). Inside each drawer are cards with the website name, your username, and your password.

Everything is scrambled with strong encryption. If someone steals the filing cabinet, they can't read anything without your key.

4. Installation — Step by Step

We use a tool called uv. It installs Python programs safely, without messing up your computer.

Step 1: Install uv

Mac or Linux

Open Terminal, paste this, press Enter:

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows

Open PowerShell, paste this, press Enter:

powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

Step 2: Create folder

uv init notion-vault-cli
cd notion-vault-cli

Step 3: Install requirements

uv add cryptography rich prompt_toolkit pyperclip

Step 4: Add the program file

Download notion_vault_cli.py and put it inside the notion-vault-cli folder.

Step 5: Run it

uv run python notion_vault_cli.py

First run: It will ask you to create a master password. This creates your vault file at ~/.pma_vault.db. That's it — you're ready.

5. Your Master Password

This is the only password you must remember. It unlocks everything.

If you forget it, no one can recover your vault. Not us, not anyone. It's designed that way for security.

Choose something long you won't forget, like blue coffee table rain 1998. Write a hint on paper, never the password itself.

8. Folders Explained

You start with three drawers. You can make more anytime just by using a new name.

  • Personal — Facebook, Instagram, Gmail, Netflix, bank
  • Education — Blackboard, Moodle, university portal, Gradescope
  • Business — work email, AWS, GitHub, company VPN

Why separate? It keeps you organized and lets you list education without seeing everything else. You can also create "Family" or "Projects" — just say save router in family.

9. Password Generator

Let it make strong passwords for you. They look like k8Pq_3vTn!xL92aQ — impossible to guess.

When you save and leave the password blank, it generates one automatically. It copies to your clipboard for 15 seconds, then clears itself.

Best practice: never make up your own passwords. Always use generate.

10. Context Memory (Facts)

This is different from passwords. Use it for things you always forget.

Examples:

  • remember my national id is 1087654321
  • remember wifi name is Home-5G
  • remember my work badge is 44219

Later, just ask what is my national id. It's stored encrypted, same as passwords.

11. Security — Plain English

Where is data stored?On your computer only, in a file called ~/.pma_vault.db
Who can see it?Only someone with your master password
Is it online?Never. Works without internet.
What if my computer is stolen?Thief cannot open the vault without your master password. It's encrypted with AES-256.
Backups?Copy the .db file to a USB drive. That's your backup.

12. Backup and Restore

Backup

  1. Close Notion Vault
  2. Find the file: ~/.pma_vault.db (Mac/Linux: in your home folder, show hidden files. Windows: C:\Users\You\.pma_vault.db)
  3. Copy it to a USB drive or external drive

Restore

  1. Install Notion Vault on new computer
  2. Copy the backup file back to the same location
  3. Run the program, enter your master password

Warning: The "export" command creates a readable text file. Only use it if you know what you're doing, and delete it after.

14. Best Practices

  1. Use a different password for each site. That's the whole point.
  2. Let Notion Vault generate passwords. Don't type "Ali123".
  3. Backup your vault monthly. Put it on a USB.
  4. Lock your computer when away. Win+L or Ctrl+Cmd+Q
  5. Never share your master password. Not even with IT.

15. Frequently Asked Questions

Q: Does it upload to the cloud?
A: No, never. Everything stays on your computer.
Q: Can I use it on my phone?
A: Not yet. It's computer only for now.
Q: What if I forget my master password?
A: You cannot recover it. You'll need to create a new vault and re-save passwords. This is for your security.
Q: Is it free?
A: Yes. No fees, no subscriptions.
Q: Can my family use the same vault?
A: Better if each person has their own. Sharing a master password defeats the purpose.
Q: Does it work in Arabic?
A: Commands are in English, but it stores Arabic passwords, usernames, and notes perfectly fine.
Q: How do I uninstall?
A: Delete the notion-vault-cli folder and delete ~/.pma_vault.db. That's all.
Q: Is it safe?
A: Yes. It uses AES-256 encryption, the same standard banks use.
